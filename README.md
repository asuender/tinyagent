# tinyagent

The tiniest possible coding agent, written in Go.

## About

> **Disclaimer**: I am new to Go, and this stands as the very first project written in it. Feel free to look through my repo, but please don't expect production-grade code here. It's just a toy project. Have fun!

With the ongoing rise of AI, people (including myself) get more and more worried about the (negative) ramifications we will experience with the continuous release of more capable LLMs. Heck, it will be pretty depressing to find ourselves de facto useless in the eyes of tech companies striving for bigger profits. Nevertheless, despite all of that, I've decided to go back to handcoding again. It's so much joy to just think.
